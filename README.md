# Поиск слов в пдф файлах

## Установка и настройка
 1. Установка всех библиотек
 ```pip install -r requirements.txt```

 2. Установка **Tesseract**
 Я использовал данную версию: https://github.com/tesseract-ocr/tesseract/releases/download/5.5.0/tesseract-ocr-w64-setup-5.5.0.20241111.exe
 Ссылка на репозиторий с программой: https://github.com/UB-Mannheim/tesseract
 После скачивания установщика, выбираем папку установки и нужные языки (В моей программе это русский + английский)
 Далее Win+R, вводим ```sysdm.cpl```, "Дополнительно" -> "Переменные среды", В "Системные переменные" найдите "Path" -> "Изменить"
 "Создать" и добавьте путь к папке установки Tesseract
 Везде прожмите "OK"

 3. Установка **Poppler**
 Я использовал версию 24.08.0, ссылка: https://github.com/oschwartz10612/poppler-windows/releases/download/v24.08.0-0/Release-24.08.0-0.zip
 Ссылка на репозиторий: https://github.com/oschwartz10612/poppler-windows
 Проделываем почти тоже самое что и в пункте **2**
 Далее Win+R, вводим ```sysdm.cpl```, "Дополнительно" -> "Переменные среды", В "Системные переменные" найдите "Path" -> "Изменить"
 "Создать" и добавьте путь к папке **bin** в папке установке (данная папка находится в Libary)
 Везде прожмите "OK"

 4. Создание нужных папок
 Создайте папки **local**, **ocr_cache** в директории проекта
